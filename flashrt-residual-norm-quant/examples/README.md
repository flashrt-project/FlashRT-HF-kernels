# examples

Run the minimal residual/RMSNorm/FP8 quant example after installing `kernels` or
after adding a built artifact to `PYTHONPATH`:

```bash
python flashrt-residual-norm-quant/examples/residual_norm_quant_block.py
python flashrt-residual-norm-quant/examples/residual_norm_quant_block.py --compile
```
